專題：眾數應用互動遊戲系統

執行方式

1. Streamlit 介面（管理/投票/分析）：`streamlit run main.py`

一、專題目標
本專題旨在透過實作互動遊戲系統，使學生理解統計學中「眾數（Mode）」的概念，
並結合 Python 程式設計能力，完成資料輸入、檔案存取、資料處理、統計分析與視覺化呈現。學生需能說明程式邏輯並理解每一個步驟的意義。

二、情境說明
請設計一個班級互動投票或遊戲系統，例如：

1. 最喜歡飲料票選系統
2. 猜拳結果統計系統
3. 最受歡迎顏色票選
4. 午餐選擇投票
   系統需能記錄多位使用者資料並分析結果。

三、任務一：互動輸入與檔案存取（30分）
請設計互動程式讓使用者輸入姓名與選項，並寫入 CSV 檔案，再讀取檔案資料。
測試輸入範例：
小明 奶茶
小華 紅茶
小美 奶茶
end
預期檔案內容：
姓名,選項
小明,奶茶
小華,紅茶
小美,奶茶
預期讀取結果：[['小明','奶茶'],['小華','紅茶'],['小美','奶茶']]

四、任務二：眾數計算（20分）
需統計每個選項出現次數並找出眾數。
範例：奶茶2次、紅茶1次 → 眾數為奶茶
並列眾數範例：奶茶2次、紅茶2次 → 眾數為奶茶與紅茶

五、任務三：統計分析（15分）
需輸出：

1. 總人數
2. 各選項次數
3. 眾數
4. 最少選項
   範例輸出：
   總人數：3
   奶茶：2次
   紅茶：1次
   眾數：奶茶
   最少選項：紅茶

六、任務四：統計圖（15分）
使用 matplotlib 繪製圖表（長條圖或圓餅圖）。
圖表需包含標題、座標軸，並能清楚呈現眾數。
例如：奶茶最高柱 → 代表眾數

七、任務五：進階任務（10分）
請選擇一項完成：

1. 即時顯示眾數
   輸入
   小明 奶茶
   小華 紅茶
   小美 奶茶

---

預期輸出
目前眾數：奶茶（1票）
目前眾數：奶茶、紅茶（各1票）
目前眾數：奶茶（2票）

2. 多輪投票比較
   測試資料
   day1：
   奶茶 3
   紅茶 2
   day2：
   紅茶 4
   奶茶 1

---

預期輸出
第一天眾數：奶茶
第二天眾數：紅茶
變化結果：紅茶成為新眾數

八、任務六：成果展示（10分）
每組需進行展示，說明：

1. 系統設計
2. 資料流程
3. 眾數計算方式
4. 圖表呈現
5. 延伸功能
